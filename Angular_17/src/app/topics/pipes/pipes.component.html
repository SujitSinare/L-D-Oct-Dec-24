Default Pipe & Custom Pipe
uppercase pipe :-
Original Value - {{demoText}}
after Pipe - {{demoText | uppercase }}
lowercase pipe :-
Original Value - {{demoText}}
after Pipe -{{demoText | lowercase }}
titlecase pipe :-
Original Value - {{demoText}}
after Pipe -{{demoText | titlecase }}
currency pipe :-
Original Value - {{amount}}
after Pipe - {{amount | currency: 'INR'}}
date pipe :-
Original Value - {{currentDate}}
after Pipe -{{currentDate |date}}
date pipe ('dd/MMM/yyyy') :-
Original Value - {{currentDate}}
after Pipe -{{currentDate |date:'dd/MMM/yyyy'}}
json pipe :-
Original Value - {{student}}
after Pipe - {{student|json}}
async pipe :-
Original Value - {{currentTime}}
after Pipe - {{currentTime | async}}
custom pipe :-
Original Value - {{student.state}}
after Pipe - {{student.state |na}}
Pipe chaining
Use of async & date pipe :-
Original Value - {{currentTime}}
after Pipe - {{currentTime | async | date:'hh:mm:ss'}}
